# ClipTube AI localized screenshots

Put localized app screenshots here using this naming pattern:

- `app-preview-en.png`
- `app-preview-es.png`
- `app-preview-pt.png`
- `app-preview-fr.png`
- `app-preview-de.png`
- `app-preview-ja.png`
- `app-preview-zh-Hans.png`
- `app-preview-ru.png`
- `app-preview-ar.png`
- `app-preview-hi.png`

The website will automatically try to load the screenshot for the active language.
If it does not exist, it will fall back to:

1. `app-preview-en.png`
2. `../app-preview.svg`
